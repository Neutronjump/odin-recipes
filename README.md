# odin-recipes
The first project on The Odin Project's curriculum.

A simple website to show you three cooking recipes in their own separate webpages.
## First rendition: Only html
This project uses basic knowledge of html to show proficiency in linking to other webpages, embedding images, using lists and different headers. 

## Second rendition: Updated with CSS
This project was updated to include CSS styling to all pages.

# Reflection
Being in total control of a project gave me a sense of freedom that I'm already yearning for again. It was difficult trying to plan out all the classes and their styling, so it felt like I was just adding classes instead of trying to edit already existing classes and their styling. 

Regardless of these challenges, this was a wonderful learning opportunity and I believe to have come out of it a better developer (Even if I'm still just barely scratching the surface)
